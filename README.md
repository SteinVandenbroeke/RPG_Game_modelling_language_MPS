# RPG_Game_modelling_language_MPS
## Getting started
First add following plugins in MPS: 
- "https://plugins.jetbrains.com/plugin/13253-mps-multiline-property-editor-component"
- "https://plugins.jetbrains.com/plugin/13255-mps-table-editor-component"
- "https://plugins.jetbrains.com/plugin/13239-mps-cell-layout"
- "https://plugins.jetbrains.com/plugin/13235-mps-various-extensions"

### RPG_DSL game modeling
To create a game model using the RPG_DSL you can create a new solution, then in the soltion a model. Then add RPGLang to the used languages of the model.
Now you can start creating your game by adding a World in the model.


### muMLU_RPG_DSL game modeling
To create a game model using the muMLU RPG model you can create a new solution, then in the soltion a model. Then add muMLU to the used languages of the model.
Now you can start creating your game by making a Model.

## Getting a runnable models.py
To create a models.py right click in your model and press "preview generated text". This can now be used in th assigment 4 runner of the MDE course.
